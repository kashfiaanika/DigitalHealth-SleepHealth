Sleep Health & Lifestyle Dashboard

Course: Digital Health   |   Author: Anika Tabassum Kashfia

Project Overview

This project explores the relationship between lifestyle factors (stress level, BMI category, physical activity, and daily steps) and sleep outcomes (sleep duration and sleep quality) using the Sleep Health and Lifestyle dataset from Kaggle. The analysis is presented through an interactive R Shiny dashboard that allows users to dynamically filter the data and explore sleep-related patterns.

Open Live Shiny App View GitHub Repository

This page serves as the HTML deliverable for the Digital Health course and provides direct access to the deployed Shiny dashboard and project source code.

Dashboard Features

  • Interactive filters for age, BMI category, and sleep disorder
  • Sleep quality vs stress level visualization
  • Sleep duration comparison across BMI categories
  • Relationship between daily steps and sleep quality
  • Distribution of sleep duration across the population

Dataset source: Kaggle — Sleep Health and Lifestyle Dataset.
